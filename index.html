Propriedade ‘display’

A propriedade display define como um elemento no HTML deve ser exibido na sua página web. Ela possui diversos atributos, cada um com um comportamento específico.

Cada elemento HTML é representado por uma caixa que contém o conteúdo do elemento e determina a quantidade de espaçamento em torno desse conteúdo. A propriedade CSS display especifica como esta caixa deve aparecer na página web em relação a outros elementos, bem como o comportamento de seus elementos filhos (ou seja, os elementos dentro dela). No CSS, um elemento pode ser um dos dois “níveis”: Block ou Inline.

Os elementos que possuem o display do tipo Block, ocupa todo o espaço da largura disponível na página web. Os elementos < div > e < p > são exemplos de elementos que por padrão são do display Block.

Já os elementos de display Inline ocupa apenas o espaço necessário para apresentar o conteudo do elemento, ou seja, possibilitando que tenha mais de um elemento de display Inline na mesma linha. < span >, < b > e < a > são exemplos de elementos desse display.

CSS display:block

O display:block torna qualquer elemento em um "bloco". Os elementos iniciam uma nova linha e abrangem toda a largura da página, como os elementos < div > se comportam. Há também quebras de linha antes e depois desses elementos.

See the Pen Untitled by Leo Almeida (@lLeoAlmeida) on CodePen.

CSS Display:inline-block

O valor inline-block é um híbrido de inline e block. Exibição de elementos atribuídos: o bloco inline aparece na mesma linha com outros elementos inline, uma característica dos elementos inline. No entanto, os elementos de bloco inline também são como elementos de bloco em que você pode alterar suas larguras e alturas com CSS.

See the Pen CSS Display: inline-block by Leo Almeida (@lLeoAlmeida) on CodePen.

CSS Display: list-item

Exibição de elementos atribuídos: item de lista se comporta como elementos < li >. O elemento inteiro torna-se um elemento de nível de bloco, o texto interno torna-se seu próprio elemento embutido e um marcador é adicionado.

CSS display:none

display:none remove o elemento selecionado (e todos os conteudos dentro dele) da página. Isso faz com que os demais elementos se comportem como se esse elemento não existisse.
Neste exemplo, display: none é aplicado ao segundo elemento < div > e ao segundo elemento e < span >.

See the Pen Display: none by Leo Almeida (@lLeoAlmeida) on CodePen.

Para ocultar um elemento sem afetar o layout da página ao inves de usar o display:none, use a propriedade de visibility: hidden;
